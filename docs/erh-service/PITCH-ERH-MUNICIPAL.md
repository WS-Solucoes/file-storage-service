# 🚀 eRH Municipal
## Sistema de Gestão de Recursos Humanos para Prefeituras

## 📋 O QUE É O eRH MUNICIPAL?

O **eRH Municipal** é um sistema completo para gerenciar todo o RH da sua prefeitura em um único lugar. Chega de planilhas, sistemas antigos e dor de cabeça com eSocial e TCE!

### ✨ O que você ganha com o eRH?

| Para a Prefeitura | Para o Setor de RH | Para o Servidor |
|-------------------|-------------------|-----------------|
| ✅ Fim das multas do eSocial e TCE | ✅ Folha processada em horas, não dias | ✅ Consulta seu contracheque online |
| ✅ Economia de até 80% com TI | ✅ Relatórios prontos com 1 clique | ✅ Solicita férias pelo celular |
| ✅ Transparência para auditorias | ✅ Menos erros, menos retrabalho | ✅ Simula aposentadoria |

## 🎯 VOCÊ RECONHECE ESSES PROBLEMAS?

### O dia a dia difícil do RH municipal

| ❌ Problema | 😰 Consequência |
|-------------|-----------------|
| Sistema antigo que trava e não tem suporte | Perde dados, atrasa a folha |
| Planilhas de Excel espalhadas | Informação desatualizada, erros |
| eSocial enviado manualmente | Multas de até **R$ 181 mil** por infração |
| TCE/SAGRES com erro | Glosa de repasses, processo no gestor |
| Servidor liga toda hora perguntando | Equipe sobrecarregada |
| Servidor local com manutenção cara | Dinheiro jogado fora |

### 💸 Quanto isso custa para o município?

- **Multas eSocial**: R$ 402 a R$ 181.284 por infração
- **Multas TCE**: Gestor pode responder pessoalmente
- **Retrabalho**: 40% do tempo desperdiçado em correções
- **Processos trabalhistas**: Erros de cálculo viram ações na Justiça
